\# 01 - Análisis de Requisitos



\## Objetivo

El cliente necesita una infraestructura web basada en la pila LAMP para alojar su presencia web y un sistema de gestión interna.



\## Requisitos Técnicos

\* \*\*Sistema Operativo\*\*: Ubuntu Server 22.04 LTS.

\* \*\*Servidor Web\*\*: Apache.

\* \*\*Lenguaje\*\*: PHP.

\* \*\*Base de Datos\*\*: MySQL/MariaDB.

\* \*\*Seguridad\*\*: Acceso SSH restringido y firewall UFW.

\* \*\*Monitorización\*\*: Netdata.
