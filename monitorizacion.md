# Monitorización del Sistema

Para que el cliente pueda ver el estado del servidor en tiempo real, instalaremos Netdata.

## Instalación
Ejecutar el script oficial:
`wget -O /tmp/netdata-kickstart.sh https://my-netdata.io/kickstart.sh && sh /tmp/netdata-kickstart.sh`

## Acceso
El panel estará disponible en el puerto 19999 (http://IP_DEL_SERVIDOR:19999).
